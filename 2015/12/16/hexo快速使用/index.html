hexo快速使用

hexo基础命令

  • hexo new “name” 新建一个博客文章
  • hexo generate 生成静态文件
  • hexo server 预览自己的博客,在浏览器里面输入http://0.0.0.0:4000/
  • hexo deploy 将本地的文件部署到你的github主页上

这四个命令是最基本的操作，都有简写形式，分别为hexo n “name”, hexo g, hexo s, hexo d

新建一个页面

首先通过hexo n “name”命令新建一个页面，在source/_posts目录下找到刚才新建的.md文件，用Mou软件打开，大概默认的是这样的:

可以编辑标题、时间、标签和内容

分类设置

发现上面所做没有分类的选项。我们可以手动加入categories:项，切记冒号后面要空一格再写内容，或者打开scaffolds/post.md文件，在tags: 上面加入categories: ，保存后，重新执行hexo n “name”,会发现新建的页面上有categories:项了
scaffolds目录下，是新建页面的模板，执行新建命令时，是根据这里的模板页完成的，所以可以在这里根据自己的需求添加一些默认值
